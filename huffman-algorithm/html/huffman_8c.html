typedef struct node node_t

  • Node used for the main operations of the algorithm

  • It is used for the linked list process
  • It is used for the binary tree process
    Parameters
    elementPoints to the element that repeats itself throughout the file used in the program
    frequencyPoints to the frequency in which said element repeats itself in the file
    nextPoints to the next node in the linked list
    leftPoints to this node's left child
    rightPoints to this node's right child
    Note
    - Create it using create_node()
    - Destroy an instance of it using destroy_node()
